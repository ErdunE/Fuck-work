# Extension Icons

For Phase 3.6, use simple placeholder icons or create basic icons using online tools.

Required sizes:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

Simple approach: Use an online icon generator or create solid color squares for testing.

## Creating Placeholder Icons

You can create simple colored squares using any image editor or online tool:
- Use a solid color like #2196F3 (blue)
- Add text "FW" in white for branding
- Export in PNG format at the required sizes

Or use base64 placeholders temporarily and replace with proper icons later.
